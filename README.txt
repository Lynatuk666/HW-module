Программа "Бухгалтерия"
в модуле people.py имеется класс employee, у которого обязательны атрибуты Имя и Название должности на котрую его взяли. дата принятия п автомату является датой создания объекта класса
у класса employee есть метод get_employees который выводит информацию о работнике, его имя, должность и сколько дней н рабоатет. 
в модуле salary имеется функция calculate_salary которая выводит информацию об общей зарпате всех сотрудников
в модуле main помимо прочего импортирован модуль Colorama  котрый позволяет изменять отображение вывода в консоли(делать цветной текст, добавлят фон под буквами)
